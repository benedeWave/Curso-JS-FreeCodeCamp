# freeCodeCamp JavaScript Journey 🚀

Este repositorio contiene mis prácticas del curso de **JavaScript Algorithms and Data Structures**. 

### 🇪🇸 Documentación en Español
A diferencia de un repositorio convencional, aquí encontrarás **comentarios propios en español** en cada archivo. Mi objetivo es:
* **Traducir la lógica:** Explicar con mis palabras conceptos como la **coerción de tipos** o el uso de **Template Literals**.
* **Documentar el proceso:** Registrar cómo resuelvo desde mini-bots hasta algoritmos complejos.
* **Refuerzo de conceptos:** Usar el código como una bitácora de estudio activa.
